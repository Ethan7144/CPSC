Code Snippet

Below is a code snippet from my Algorithms class, this is written in C#.

                static int FirstFit(int[] weight, int maxWeight)
                {
                    int bin = 1, binSize = maxWeight;
                    Array.Sort(weight);
                    for (int i = 0; i < weight.Length; i++)
                    {
                        if (binSize - weight[i] < 0)
                        {
                            bin++;
                            binSize = maxWeight;
                            binSize = binSize - weight[i];
                        }
                        else
                            binSize = binSize - weight[i];
                    }
                    return bin;
                }

What this code does it is creating a firstfit alogrithm for the bin packing problem. First Fit is a Heuristic Solution as it will solve the problem and give the total number of boxes needed at a much faster rate, but it will not be as efficient of a solution when comparing to a heuristic solution. This code will return the total number of bins needed to fit n boxes that will fill k size of the box.
