CPSC 4710, Security in Computing

Class description

The class teaches the fundamentals of cybersecurity. We will be learning how to keep ourselves safe on the internet, and also how to keep others safe. We will be learning techniques on things like how to create malware and how to counter them as well.

Techniques

  • Sniffing, using a packet to try to read sensetive data.
  • Password Cracking, we will learn how to crack passwords for accounts
  • Malware, we will be learning about the dangers of them, how to counter them, and how to make them
