CPSC 3200 Object Oriented Design

Class description

The class teaches the fundamentals of object oriented design in coding. It is one of the essiential classes needed in order to program efficiently in this major, and is one of the major requirements to graduate from college.

Techniques

  • Inheritence, which is a class derives from another class
  • Composition, a technique to derive a has-a relationship between classes
  • Polymorphism, describes situations which happens in different forms
