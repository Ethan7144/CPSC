CPSC 3500, Computing Systems

Class description

The class teaches the basics about software and some hardware capabilities of computers. We learn about things regarding the OS, how ports work, and how to be able to tell what IP is acceptable. It teaches the most basics for IT concepts and also some algorithms are touched, such as dijkstras alogrithm.

Techniques

  • Operating Systems, learned about what they do and how they work
  • Dijkstras, a technique to be able find the shortest path, it is a shortest path algorithms
  • Multithreading, enables the execution of multiple parts of the program at the same time.
